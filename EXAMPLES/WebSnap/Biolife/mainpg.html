ISAPI DLL notes:

To allow the BDE to work with IIS 5 on Windows 2000, you must do the following:
You must give the IUSR_MACHINE_NAME and IWAM_MACHINE_NAME accounts full read and write permissions to the following directories:
1. The BDE directory
2. The TSession.NetDir and TSession.PrivateDir set on your Session component
3. Your actual temp directory, probably C:\WINNT\Temp
4. Your directory that contains the database files
5. If all else fails, give read and write permissions to C:\WINNT.
