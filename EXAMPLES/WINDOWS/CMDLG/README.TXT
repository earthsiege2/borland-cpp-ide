TITLE:
    CMDLG

PURPOSE:
    Demonstrates the use of Open, Color, and Font common dialogs which are
    available in Windows 3.1, Windows NT, and Windows 95.

COMMENTS:
	 The following examples also use common dialogs:
	 \BC5\EXAMPLES\SERIES\COMMONDG
	 \BC5\EXAMPLES\WIN95\COMDLG32
	 \BC5\EXAMPLES\WINDOWS\TRUETYPE
