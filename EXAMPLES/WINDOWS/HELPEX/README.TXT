TITLE:
    HELPEX

PURPOSE:
	 Illustrates various methods of presenting information in a Windows
	 help file.  Help project (hpj) and source (rtf) are provided.  The
	 application HELPEX detects F1 keystrokes to provide context sensitive
	 help for menu items.
