There are two example program that can be of great assistance
to the developer. Open the ClassBrw and the WinTools examples
separately and do a Build. After each .exe is created:

1) Select the Tools|Configure Tools menu item. 
2) Select Add
3) Under Title type VCL Object Browser 
4) Browse for the program under the BCB 3.0\Examples\ClassBrw\ObjBrow.exe
5) Select OK and then Close
6) Repeat the above for the WinTools example

Both tools should now appear under the Tools menu item.
