Title: WING Example

Keywords: TWinG;WinG example

The WING example illustrates how to use the TWinG class which
encapsulates the WinG graphics library. Note that to run this example,
you will need to have the WinG DLLs installed on your system. If you do
not have WinG, please contact Microsoft for a copy.
