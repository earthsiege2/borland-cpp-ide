TITLE:
    CURSORS

DESCRIPTION:
    Change your cursor's shape by selecting from a list of cursor constants.

DEMONSTRATES:
    + TCursor, TComboBox.
    + Using cursor constants to change the shape of a cursor.
    + Responding to the OnChange event of a ComboBox.

COMMENTS:
    For a demonstration of using cursors other than those provided through
    VCL, see the SWAT example.
